# My Xfce-4 Dotfile
## _My setup config_
    OS: openSUSE Tumbleweed x86_64 
    Kernel: 5.18.6-1-default 
    Shell: bash 5.1.16 
    WM: Xfwm4
## _Installation_
#### Clone this repository
    git clone https://github.com/bembenk18/dotfile
#### Backup your config
    mv /home/\$USER/.config/ /home/$USER/.config-backup
#### Move a new config
    mv dotfile/.config/ /home/$USER/TES
